## Módulo 2

Segunda atividade do módulo:<br/>
Fazer um código com 3 valores e encontrar o maior valor.<br/>
[Arquivos da Atividade](https://github.com/AlanLee1/Mentorama-Csharp/tree/main/Files/Module2)

```
int a = 26;
int b = 100;
int c = 30;
int maior = 0;

if (a > b)
{
    maior = a;
} else
{
    maior = b;
}
if (c > maior)
{
    maior = c;
}

Debug.Log(maior);
```
